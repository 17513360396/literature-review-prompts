## 概述

这是 **RIKEN 国际数据同化学校（RISDA 2018）** 的教学材料——相当于数据同化领域的"标准教材配套代码"。它使用 Lorenz-63 混沌模型作为教学平台，从最优插值到粒子滤波，**逐方法对比**，非常适合理解不同 DA 方法之间的优劣。

## 为什么用 Lorenz-63

Lorenz-63 是经典的 3 维混沌系统：

```
dx/dt = σ(y - x)
dy/dt = x(ρ - z) - y
dz/dt = xy - βz
```

选择它作为教学模型的原因：
- **低维**（3 维）→ 所有矩阵都可以显式查看/打印
- **混沌**（对初值敏感）→ 必须有同化才能维持预报精度
- **非线性**→ 能演示 EKF 线性化误差、EnKF 非线性传播的优势
- **有"真值"**→ 可以客观评估同化效果

## 最优插值 Notebook 内容

### OI 在 Lorenz-63 上的实现

```python
# 1. 定义背景场 x_b（来自模式的短期预报）
# 2. 生成模拟观测 y（真值 + 噪声）
# 3. 指定背景误差协方差 B（通常取气候态方差）
# 4. 指定观测误差协方差 R（根据仪器精度）
# 5. 计算 Kalman 增益
K = B @ H.T @ np.linalg.inv(H @ B @ H.T + R)
# 6. 计算分析场
x_a = x_b + K @ (y - H @ x_b)
```

### 关键实验

1. **观测密度的影响** — 每步同化 vs 每 8 步同化
2. **观测精度的影响** — 改变 R 的大小
3. **背景误差协方差的敏感度** — B 过大/过小的影响
4. **与 Nudging 的对比** — OI 在数学上更优

## 核心收获

在这个教程中你会学到的**通用原则**（可以直接搬到任何真实 DA 场景中）：

| 原则 | 说明 |
|:---|:---|
| **B 的尺度很重要** | B 太大 → 过于信任观测（分析场噪声大）；B 太小 → 不信任观测（背离真值） |
| **R 的尺度很重要** | 过小的 R → 过度拟合观测噪声；过大的 R → 浪费观测信息 |
| **OI 优于 Nudging** | OI 是最小方差无偏估计，Nudging 只是启发式方法 |
| **观测频率不影响一致性** | 只要 B 和 R 正确指定，OI 在任何观测频率下都是最优的 |
| **静态 B 的局限** | OI 的 B 不随时间演变，在流型剧变时预报误差会系统性低估 |
