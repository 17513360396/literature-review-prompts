## 5 条推荐学习路径

### 路径 A：最快上手（1 天）

```
上午: Karpathy 视频 "Let's build GPT" (2小时)
下午: nanoGPT shakespeare_char 示例跑通 (3小时)
晚上: 用 nanoGPT 在你自己的文本上训练
```
→ 你会得到：一个能训练 GPT 的工作环境 + 对 Transformer 的直观理解

### 路径 B：从论文到代码（3 天）

```
Day 1: 读 "Attention Is All You Need" 论文
       + Jay Alammar "Illustrated Transformer" 博客

Day 2: Annotated Transformer Notebook 逐行阅读
       + 在 Colab 中修改参数、观察变化

Day 3: haukzero/Transformer-Demo 逐个文件走读
       + 自己手写一遍 Attention 函数
```
→ 你会得到：对原始 Transformer 架构的深入理解

### 路径 C：中文母语路线（2-5 天）

```
Day 1: art1524/learn-nlp-with-transformers (图解理解)
Day 2: annotated-transformer-Chinese (论文对照)
Day 3: haukzero/Transformer-Demo (代码详读)
Day 4-5: Datawhale/fun-transformer (NumPy 手写一遍)
```
→ 你会得到：用母语完整掌握 Transformer 的每个细节

### 路径 D：LLM 工程师路线（2-4 周）

```
Week 1: rasbt/LLMs-from-scratch Ch2-4
        (tokenization, attention, GPT architecture)
Week 2: rasbt/LLMs-from-scratch Ch5 + nanoGPT 训练
        (预训练 + 微调)
Week 3: mlabonne/llm-course
        (SFT, RLHF, DPO)
Week 4: HuggingFace Course + 自己的项目
```
→ 你会得到：能从零构建和训练 LLM 的工程能力

### 路径 E：广度优先（1 周，适合决策者）

```
Day 1: minGPT model.py 精读 (理解 GPT 有多简单)
Day 2: ViT 教程 (理解 Transformer 在视觉中的应用)
Day 3: iTransformer (理解 Transformer 在时序中的应用)
Day 4: HuggingFace Course Ch2-3 (理解应用层)
Day 5: attnhut (浏览所有注意力变体)
```
→ 你会得到：对 Transformer 生态的全景理解

## 核心概念掌握清单

- [ ] **Self-Attention 公式**：`softmax(QKᵀ/√d_k)·V`，能手写推导
- [ ] **Multi-Head Attention**：理解为什么分头 + 拼接
- [ ] **Q, K, V 的来源**：从同一输入经过不同线性投影得到
- [ ] **Positional Encoding**：正弦 vs 可学习
- [ ] **残差连接 + LayerNorm**：Pre-Norm vs Post-Norm
- [ ] **因果掩码**：Decoder 为什么需要 + 如何实现
- [ ] **Cross-Attention**：Encoder-Decoder 架构的核心
- [ ] **FFN (Position-wise Feed-Forward)**：两层 MLP + 激活函数
- [ ] **Tokenization**：BPE vs WordPiece vs SentencePiece
- [ ] **训练技巧**：学习率 warmup、梯度裁剪、混合精度
